# 230 Coursework A2

Coursework for 230

## How to compile

From root directory run:

    mkdir bin
    javac -d bin -sourcepath src src/Main.java
    cp -r src/res bin/res
    cp -r src/fxml bin/fxml

You can now run the program by:

    java -classpath bin.Main

*note don't commit class files **EVER***
